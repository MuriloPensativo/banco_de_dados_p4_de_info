1 - sistema isolado: Quando os dados de cada setor do seu sistema de dados são salvos em lugares diferentes e sem compartilhamento dos dados, temos um sistema isolado, onde cada setor é responsável por cuidar dos próprios dados, um exemplo é o setor de venda e produção de produtos, os dados de cada coisa vendida ficaria no setor de vendas e os dados de cada coisa produzida ficaria no setor de produção que salvaria e organizaria de forma independente os dados, o que vai levar a redundância e outros problemas
2 - Sistema integrado: quando os dados do seu sistema são salvo de forma compartilhada, ou seja, cada setor tem acesso aos dados de outro setor e alteram os dados de forma que a mudança aconteça igualmente para todos os outros setores, podemos fazer isso salvando todos os dados em um único lugar em que todos os setores tenham a capacidade de mexer, isso é o conceito de banco de dados.
